
Projet d'upload de fichiers en PHP.

## Installation

1. Cloner le repo
2. Lancer avec XAMPP / WAMP
3. Accéder à `index.php` via le navigateur


## Configuration

Tout se modifie dans `config.php` :
- Nom du développeur
- Couleurs
- Types de fichiers autorisés
- Taille max des fichiers


## Fonctionnalités

- Upload de fichiers (PDF, JPG, PNG, GIF)
- Taille max : 5Mo
- Messages d'erreur / succès
